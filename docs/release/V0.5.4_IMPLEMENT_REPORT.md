## 4. TeachingTask 新边界定义

本次实施完成后，排课系统中"教学任务合班"能力被明确限定为：

> **仅支持显式合班**：所有多班绑定通过创建/编辑表单的"班级"多选或 Excel 导入行的 `班级代码=CS2301,CS2302` 完成。
>
> **不支持自动合班**：不再存在"按课程名+教师名推测"的自动检测路径，消除了"不同学时任务被错误合并"的根因。
>
> **数据模型不变**：`TeachingTask ↔ TeachingTaskClass ↔ ClassGroup` 多对多结构完整保留，支持一个任务绑定任意数量的班级。
