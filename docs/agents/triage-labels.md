# Triage Labels

工程技能使用五个标准 triage 角色。本文件将这些角色映射到项目实际使用的标签字符串。

| 技能中的标签               | 本项目标签   | 含义                               |
| -------------------------- | ------------ | ---------------------------------- |
| `needs-triage`             | `待评估`     | 维护者需要评估这个 issue           |
| `needs-info`               | `待补充`     | 等待提交者补充更多信息             |
| `ready-for-agent`          | `可分配`     | 规格完备，可由 AI Agent 认领       |
| `ready-for-human`          | `需人工`     | 需要人工实现                       |
| `wontfix`                  | `不处理`     | 不予处理                           |

当 skill 提到某个角色时（如"应用可分配标签"），使用本表右侧对应的标签字符串。
