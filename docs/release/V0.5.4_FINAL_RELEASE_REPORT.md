## Features

### 1. TeachingTask Domain Stabilization

**删除**：
- `MergeableGroup` struct（DTO）
- `TeachingTaskService.DetectMergeableTasks` 方法（按课程名+教师名分组检测）
- `TeachingTaskService.SplitMergedTeachingTask` 方法（自动拆班）
- 前端"智能检测合班"按钮及结果面板
- 前端操作列"拆班"按钮

**保留**：
- `TeachingTask ↔ TeachingTaskClass ↔ ClassGroup` 多对多关联完整保留
- `CreateTeachingTask(task, classGroupIDs []uint)` 支持手动多班绑定
- Excel 导入支持 `班级代码=CS2301,CS2302` 多班一行
- Solver 消费 `tt.Classes` 的逻辑不受影响

### 2. Seed Idempotency Stability Fix

**发现问题**：重复启动应用报 `UNIQUE constraint failed: class_groups.code`

**根因**：`seedIfAbsent` 使用 `Count` + `Create` 两步操作——若某行 Create 失败，GORM error state 传播使后续 Count 不执行，连锁失败。

**修复后**：
- `DB` 接口新增 `FirstOrCreate(dest, conds...)`
- `seedIfAbsent` 改为 `FirstOrCreate`（原子查找或创建）
- `seedTeachingTasks` 改为 `FirstOrCreate`（含 TeachingTaskClass）
- `seedDemoEntries` 增加 error 检查
