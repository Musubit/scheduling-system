## 一、这个系统能帮你做什么

### 你以前的烦恼

- 手动排课要花好几天，还要反复检查冲突
- 教师时间对不上、教室容量不够、班级课表撞车
- 排好之后一改就要牵一发动全身
- 领导要看不同角度的课表，每次都要重新整理

### 这个系统怎么帮你

**自动排课**：你只需要把教师、教室、课程、班级这些信息录入系统，告诉系统哪些课要上、谁来上、给哪个班上，系统就能自动帮你排出一整学期的课表。

**智能优化**：系统会自动考虑教师的偏好时间、课程分散度、教室容量等各种因素，尽量排出高质量的课表。

**冲突检测**：系统会自动检查教师、教室、班级的时间冲突，告诉你哪里有问题。

**多角度查看**：可以按教师看、按班级看，随时切换视角。

**版本管理**：每次排课结果都能保存下来，方便对比不同方案。

**一键导出**：排好的课表可以导出成 Excel 或 PDF，直接发给领导或教师。

## 二、第一次使用前的准备

### 2.1 获取软件

本系统是**便携版软件**，不需要安装，解压就能用。

**获取方式：**
- 从管理员处获取 `scheduling-system-portable-v*.zip` 压缩包

**使用步骤：**
1. 把压缩包解压到任意文件夹（建议放在 D 盘或 E 盘，路径用英文）
2. 打开解压后的文件夹
3. 双击 `scheduling-system.exe` 即可启动

**文件夹结构：**
```
解压后的文件夹/
├── scheduling-system.exe    ← 双击这个启动程序
└── scheduler/               ← 内部文件，不需要管
```

**小提示：**
- 可以把文件夹放在桌面、D 盘、U 盘等任何地方
- 可以创建 `scheduling-system.exe` 的快捷方式放到桌面，方便下次打开
- 首次启动可能需要等待几秒，系统会自动准备运行环境

### 2.2 首次启动

第一次打开系统时，里面是空的，你需要先录入基础数据。

**数据说明：**
- 程序和数据是分开的，删除或移动程序文件夹不会影响数据
- 更新版本时，只需用新版覆盖旧版程序，数据自动保留
- 如需备份或迁移数据，使用「系统设置」里的「导出备份」和「导入」功能

**你需要准备的信息：**

| 数据类型 | 需要的信息 | 举例 |
|---------|-----------|------|
| 教师 | 工号、姓名、所在院系 | 张三，工号 T001，计算机学院 |
| 教室 | 编号、名称、所在教学楼、楼层、容量、类型 | A101，教学楼A栋1楼，60人，普通教室 |
| 课程 | 编号、名称、学分、课时、类型 | CS101，程序设计基础，3学分，48课时 |
| 班级 | 编号、名称、年级、人数 | 计科2023-1班，大二，35人 |
| 教学任务 | 哪个教师给哪个班级上什么课、一周几次 | 张三 给 计科2023-1班 上 程序设计基础，一周2次 |

## 三、核心操作流程

整个排课过程可以分为 **6 个步骤**：

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  1.设置学期  │ →  │ 2.录入资源  │ →  │ 3.创建任务  │
└─────────────┘    └─────────────┘    └─────────────┘
                                              ↓
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  6.导出课表  │ ←  │ 5.手动微调  │ ←  │ 4.自动排课  │
└─────────────┘    └─────────────┘    └─────────────┘
```

**简单来说：**
1. 先设置好当前是哪个学期
2. 把教师、教室、课程、班级的信息录进去
3. 告诉系统"谁给谁上什么课"
4. 点一下"自动排课"，等系统算完
5. 看看结果，有不满意的地方手动调一下
6. 满意了就导出课表

## 四、详细操作指南

### 4.1 管理学期

**为什么要先设置学期？**

因为所有的课、教学任务都是按学期来管理的。你需要告诉系统当前是哪个学期，系统才能正确地帮你排课。

**操作步骤：**

1. 点击左侧菜单的「**设置**」
2. 找到「**学期管理**」区域
3. 点击「**新建学期**」按钮
4. 填写信息：
   - **学年**：比如 `2025-2026`
   - **学期**：第一学期（秋季）或第二学期（春季）
   - **开学日期**：选择这个学期的第一天
5. 点击「保存」

**如何设置当前学期？**

在页面顶部的工具栏，有一个学期选择器，点击选择你要排课的学期即可。

**小提示：**
- 系统会根据当前月份自动推荐下一个学期
- 3-8月推荐秋季学期，9-次年2月推荐春季学期

---

### 4.2 管理资源

资源就是排课需要的"原材料"：教师、教室、课程、班级。

#### 4.2.1 添加教师

1. 点击左侧菜单的「**资源管理**」
2. 默认显示「教师」标签页
3. 点击「**添加教师**」按钮
4. 填写信息：
   - **工号**：教师的唯一编号
   - **姓名**：教师姓名
   - **院系**：教师所在的学院或部门
   - **状态**：启用或停用（停用的教师不会参与排课）
5. 点击「保存」

**批量导入：**
如果教师人数很多，可以点击「**导入**」按钮，下载 Excel 模板，填好后上传即可。

#### 4.2.2 添加教室

1. 切换到「**教室**」标签页
2. 点击「**添加教室**」按钮
3. 填写信息：
   - **编号**：教室的唯一编号
   - **教室名**：比如 A101、B201
   - **教学楼**：教室所在的楼栋
   - **楼层**：在几楼
   - **容量**：能坐多少人
   - **教室类型**：
     - 普通教室 — 一般上课用
     - 多媒体教室 — 有投影、音响
     - 实验室 — 做实验用
     - 机房 — 有电脑
     - 体育场馆 — 上体育课
4. 点击「保存」

**小提示：**
- 教室类型很重要，系统会根据课程类型自动匹配合适的教室
- 比如体育课会自动安排到体育场馆，实验课会安排到实验室

#### 4.2.3 添加课程

1. 切换到「**课程**」标签页
2. 点击「**添加课程**」按钮
3. 填写信息：
   - **编号**：课程的唯一编号
   - **课程名**：比如"程序设计基础"
   - **院系**：开课的学院
   - **学分**：这门课多少学分
   - **课时**：总共多少节课
   - **类型**：必修/选修/公共课等
   - **类别**：理论课/实验课/体育课等
4. 点击「保存」

#### 4.2.4 添加班级

1. 切换到「**班级**」标签页
2. 点击「**添加班级**」按钮
3. 填写信息：
   - **编号**：班级的唯一编号
   - **班级名称**：比如"计科2023-1班"
   - **院系**：所属学院
   - **年级**：大一/大二/大三/大四
   - **人数**：这个班有多少学生
4. 点击「保存」

---

### 4.3 设置教学任务

**什么是教学任务？**

教学任务就是告诉系统："哪个教师 给 哪个班级 上 什么课"。

比如："张三老师 给 计科2023-1班 上 程序设计基础，一周上2次，每次2节课"

**操作步骤：**

1. 切换到「**教学任务**」标签页
2. 点击「**添加教学任务**」按钮
3. 填写信息：
   - **课程**：选择要上的课
   - **教师**：选择谁来上
   - **班级**：选择给哪个班上（可以选择多个班级合班上课）
   - **周课时**：一周上几次
   - **每次节数**：一次上几节课（1节、2节或3节）
4. 点击「保存」

**批量导入：**
教学任务通常很多，建议用 Excel 批量导入：
1. 点击「**导入**」按钮
2. 下载 Excel 模板
3. 按照模板格式填写所有教学任务
4. 上传填好的 Excel 文件

**小提示：**
- 教学任务是排课的核心，一定要填准确
- 如果一个教师给同一个班上不同的课，需要创建多个教学任务
- 合班上课只需要创建一个教学任务，选择多个班级即可

### 4.4 自动排课

这是系统最核心的功能。所有资源和教学任务准备好后，就可以开始自动排课了。

**操作步骤：**

1. 点击左侧菜单的「**排课**」
2. 你会看到排课配置页面

#### 第一步：选择排课学期

在页面顶部选择你要排课的学期。

#### 第二步：选择排课范围

- **全校所有院系**：给全校排课
- **某个院系**：只给某个学院排课

#### 第三步：选择排课模式

- **完整排课**（推荐）：同时安排时间和教室
- **仅排时间**：只安排时间，不分配教室（适合先确定时间再分配教室的场景）

#### 第四步：设置约束条件

约束条件就是你希望课表满足的要求。系统会按照这些要求来优化课表。

| 约束条件 | 说明 | 建议 |
|---------|------|------|
| 教师偏好时段 | 有些教师希望上午上课或下午上课 | 建议开启 |
| 课程分散度 | 同一门课尽量分散在不同天 | 建议开启 |
| 教师到校天数 | 教师尽量集中在几天上课 | 建议开启 |
| 优先低楼层 | 低年级学生尽量安排在低楼层教室 | 可选 |
| 体育课时段 | 体育课尽量安排在下午 | 建议开启 |
| 避开周六 | 不在周六排课 | 建议开启 |
| 避开周日 | 不在周日排课 | 建议开启 |

每个约束条件都有一个**权重滑块**：
- 权重越高，系统越重视这个要求
- 如果某个条件对你特别重要，可以把权重调高

#### 第五步：设置锁定时段

有些时间段你不想排课（比如午休时间、晚自习时间），可以在时间格子里点击标记为"锁定"。

#### 第六步：开始排课

点击「**开始自动排课**」按钮，系统开始工作。

**排课过程中你会看到：**
- 进度条显示当前进度
- 阶段提示：初始化 → 优化中 → 后处理 → 完成
- 一般 1-3 分钟就能完成

**排课完成后，系统会告诉你：**
- 多少门课成功排入
- 有没有冲突
- 课表质量评分（满分 100 分）

### 4.5 查看课表

排课完成后，点击「**查看课表**」或左侧菜单的「**课表**」，就能看到排好的课表。

#### 4.5.1 周视图

这是最常用的查看方式，显示一周的课表：

```
         周一    周二    周三    周四    周五    周六    周日
第1-2节   课程A   课程B         课程A   课程B
第3-4节          课程C   课程D          课程C
...
```

**切换视角：**
- 点击顶部的「**教师**」或「**班级**」按钮切换视角
- 选择具体的教师或班级，就能看到对应人的课表

**拖拽调整：**
如果觉得某个课安排得不好，可以直接用鼠标拖动课程到其他时间格子。

#### 4.5.2 时间线视图

以时间轴的方式显示，更直观地看到每天的课程安排。

#### 4.5.3 月视图

以月历的方式显示，可以看到整个月的课程分布。

### 4.6 手动调整课表

自动排课的结果可能不完全符合你的要求，你可以手动调整。

**如何调整：**

1. 在课表页面，直接用鼠标**拖动**课程块到你想要的时间格子
2. 系统会自动检查是否有冲突：
   - 如果有冲突，会弹出提示告诉你冲突原因
   - 如果没有冲突，课程就会移到新位置

**调整后会显示：**
- 左下角会显示"已调整 X 次"
- 提醒你保存调整后的结果

**小提示：**
- 手动调整后，原来的自动排课结果不会丢失
- 你可以随时保存当前状态，然后继续调整

### 4.7 保存与版本管理

#### 4.7.1 为什么要保存版本？

- 你可能想对比不同排课方案的优劣
- 领导可能要看多个备选方案
- 调整后想保留调整前的版本作为备份

#### 4.7.2 保存当前版本

1. 在课表页面，点击「**保存版本**」按钮
2. 输入版本名称（比如"方案一：教师优先"或"2026秋季排课初版"）
3. 点击「保存」

#### 4.7.3 查看历史版本

1. 点击左侧菜单的「**版本管理**」
2. 左侧列表显示所有保存过的版本
3. 点击某个版本，右侧会显示这个版本的详细信息

#### 4.7.4 对比版本

1. 在版本管理页面，选择一个版本
2. 切换到「**对比**」标签
3. 选择另一个版本进行对比
4. 系统会显示两个版本的差异

#### 4.7.5 恢复历史版本

如果想回到之前的某个版本：
1. 在版本管理页面，选择要恢复的版本
2. 点击「**恢复此版本**」按钮
3. 确认后，当前课表会恢复到那个版本的状态

---

### 4.8 导出课表

排好的课表可以导出给其他人查看。

#### 4.8.1 导出为 Excel

1. 在课表页面，点击「**导出**」按钮
2. 选择「导出 Excel」
3. 选择导出范围：
   - **当前视角**：只导出你正在看的这个教师或班级的课表
   - **全部**：导出所有教师或班级的课表
4. 选择保存位置，点击「保存」

Excel 文件可以直接用 WPS 或 Microsoft Office 打开。

#### 4.8.2 导出为 PDF

1. 点击「**导出**」按钮
2. 选择「导出 PDF」
3. PDF 文件适合打印或发给领导审阅

## 五、常见问题解答

### Q1：如何更新到新版本？

1. 从管理员处获取新版压缩包 `scheduling-system-portable-v*.zip`
2. 关闭正在运行的系统
3. 用新版文件夹覆盖旧版文件夹
4. 重新双击 `scheduling-system.exe` 启动

**注意：** 你的数据不会丢失，因为数据保存在电脑的其他位置（`C:\Users\你的用户名\AppData\Local\scheduling-system\`）。

### Q2：如何备份数据？

系统提供了备份功能，在「**系统设置**」页面的「**数据管理**」区域：

1. 点击左侧菜单的「**系统设置**」
2. 找到「**数据管理**」区域
3. 点击「**导出备份**」按钮
4. 选择保存位置，给文件起个名字
5. 点击「保存」

备份文件是一个 `.db` 文件，建议保存到 U 盘或网盘。

**恢复备份：**
1. 在「数据管理」区域点击「**导入**」按钮
2. 选择之前备份的 `.db` 文件
3. 确认后，系统会用备份数据覆盖当前数据

**注意：** 导入会覆盖当前所有数据，请谨慎操作！

### Q3：如何迁移到其他电脑？

1. 在旧电脑上：进入「系统设置」→「数据管理」→ 点击「导出备份」
2. 把备份文件（`.db`）复制到新电脑（U 盘、网盘都行）
3. 在新电脑上：解压程序，打开系统
4. 进入「系统设置」→「数据管理」→ 点击「导入」
5. 选择备份文件，确认导入

所有数据（教师、教室、课程、教学任务、排课结果）都会完整迁移。

### Q4：首次启动时弹出安全警告怎么办？

由于软件没有数字签名，Windows 可能会弹出安全警告：

1. 点击「**更多信息**」
2. 点击「**仍要运行**」

这是正常现象，软件是安全的。

### Q5：排课完成后，有几门课没有排进去怎么办？

**原因可能是：**
- 教室不够用（容量或类型不匹配）
- 教师时间冲突
- 锁定时段太多，可用时间不够

**解决方法：**
1. 查看排课结果中的"未排入课程"列表
2. 系统会告诉你每门课没排进去的原因
3. 根据原因调整：
   - 如果是教室不够 → 添加更多教室
   - 如果是教师冲突 → 调整教学任务
   - 如果是时间不够 → 减少锁定时段

### Q6：排课评分低是什么意思？

评分满分 100 分，表示课表的质量：
- **80 分以上**：很好，课表质量很高
- **60-80 分**：一般，有些地方可以优化
- **60 分以下**：较差，建议调整约束条件重新排课

评分低通常是因为约束条件之间有冲突，系统无法同时满足所有要求。

### Q7：如何让某个教师的课集中在某几天？

在排课配置中，开启「教师到校天数」约束，并把权重调高。系统会尽量让教师的课集中在几天内。

### Q8：如何让体育课安排在下午？

在排课配置中，开启「体育课时段」约束，系统会自动把体育课安排到下午时段。

### Q9：排课过程中可以取消吗？

可以。点击「**停止**」按钮，排课会立即停止，但已经排好的部分会保留。

### Q10：手动调整课表后，如何保存？

调整完成后，点击「**保存版本**」按钮，给当前状态起个名字保存即可。

### Q11：如何给多个班级合班上课？

在创建教学任务时，「班级」字段可以选择多个班级，系统会自动把它们安排在同一时间、同一教室上课。

### Q12：导入 Excel 时提示格式错误怎么办？

1. 确保使用系统提供的模板
2. 检查必填字段是否都填了
3. 检查数据格式是否正确（比如日期格式、数字格式）
4. 如果有重复数据，系统会提示冲突

## 六、术语说明

| 术语 | 通俗解释 |
|------|---------|
| **教学任务** | 告诉系统"谁给谁上什么课"的一条记录 |
| **排课** | 把教学任务安排到具体的时间和教室 |
| **约束条件** | 你希望课表满足的要求（比如教师偏好、课程分散等） |
| **硬约束** | 必须满足的条件（比如同一个教师不能在同一时间上两门课） |
| **软约束** | 尽量满足的条件（比如教师希望上午上课） |
| **锁定时段** | 你不希望排课的时间段（比如午休、晚自习） |
| **版本** | 某一次排课结果的快照，可以保存和对比 |
| **快照** | 和"版本"意思相同，保存某个时间点的课表状态 |
| **评分** | 系统对课表质量的打分，满分 100 |
| **冲突** | 同一个教师/教室/班级在同一时间被安排了多门课 |
| **排课范围** | 这次要给哪些院系排课 |
| **排课模式** | 完整排课（时间+教室）或仅排时间 |
