# AGENTS.md

Instructions for AI coding agents working in this repository.

## Agent skills

### Issue tracker

Issues 以本地 Markdown 文件形式存放在 `.scratch/<feature-slug>/` 目录下。详见 `docs/agents/issue-tracker.md`。

### Triage labels

使用中文 triage 标签：待评估、待补充、可分配、需人工、不处理。详见 `docs/agents/triage-labels.md`。

### Domain docs

单上下文模式 — `CONTEXT.md` + `docs/adr/` 位于仓库根目录。详见 `docs/agents/domain.md`。
